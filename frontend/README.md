# Spam Filter Frontend

This is a Vite + React + Tailwind frontend for the Spam Filter project.

Install and run:

1. cd frontend
2. npm install
3. npm run dev

The app will run on http://localhost:5173 and calls the backend at http://localhost:8000/api/predict by default.

To customize backend URL, set environment variable in `.env` or use Vite env key `VITE_API_URL`.
